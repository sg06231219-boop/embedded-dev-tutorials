⌨️ 教程18: USB HID键盘鼠标

📋 项目概述

本教程将带你实现一个基于STM32 USB HID的键盘鼠标设备。通过按键触发，STM32可以模拟键盘按键输入和鼠标移动操作，插入电脑后自动识别，无需安装任何驱动。

什么是HID？HID (Human Interface Device) 即人机接口设备，包括键盘、鼠标、游戏手柄等。Windows/Mac/Linux都内置了HID驱动，所以HID设备可以即插即用，无需额外安装驱动程序。

🔧 硬件准备

元件数量说明
STM32F103C8T6 最小系统板1必须有USB接口(PA11/PA12)
Micro-USB线1数据线(不是纯充电线)
轻触按键4PA0~PA3，分别映射不同按键
10KΩ 电阻4按键上拉电阻
1.5KΩ 电阻1USB 1.5K上拉(仅F103需要，部分板载)
⚠️ 关键注意事项：
1. STM32F103C8T6有USB全速(12Mbps)接口，引脚固定为PA11(DM)和PA12(DP)
2. 部分开发板(Blue Pill)已经板载了1.5K上拉电阻到PA12，可以省略
3. 使用USB线必须确认支持数据传输，不是只能充电的线
4. STM32F407使用USB OTG接口，引脚不同(需要使用CubeMX自动配置)

📖 知识讲解

USB HID协议栈架构

┌─────────────────────────────┐
│        应用层               │
│  (按键检测 → 发送HID报表)   │
├─────────────────────────────┤
│     USB HID Class           │
│  (报表描述符, 端点管理)     │
├─────────────────────────────┤
│     USB Device Core         │
│  (枚举, 描述符, 控制传输)   │
├─────────────────────────────┤
│     USB PHY (硬件)          │
│  (PA11/PA12, USB收发器)     │
└─────────────────────────────┘

USB枚举过程:
1. 主机检测到D+上拉 → 全速设备
2. 主机发送SETUP包 → 获取设备描述符
3. 主机获取配置描述符 → 知道有几个接口和端点
4. 主机获取HID报表描述符 → 知道数据格式
5. 枚举完成，设备可用

HID报表格式

类型大小说明
键盘报表8字节Modifier(1B)+Reserved(1B)+Keys(6B)
鼠标报表4字节Buttons(1B)+X(1B)+Y(1B)+Wheel(1B)
键盘报表格式(8字节):
┌────────┬──────────┬──┬──┬──┬──┬──┬──┐
│ 修饰键 │  保留    │K1│K2│K3│K4│K5│K6│
└────────┴──────────┴──┴──┴──┴──┴──┴──┘
  Modifier: 0x00=无, 0x02=Shift, 0x04=Ctrl, 0x01=Left Ctrl...

鼠标报表格式(4字节):
┌────────┬─────┬─────┬──────┐
│ Buttons │  X  │  Y  │Wheel │
└────────┴─────┴─────┴──────┘
  Buttons: Bit0=左键, Bit1=右键, Bit2=中键
  X/Y: 相对移动量(-127~127)

💻 完整代码

USB HID键盘鼠标 (main.c)

/* ============================================
 * USB HID 键盘鼠标 - 完整示例
 * MCU: STM32F103C8T6
 * 功能: 按键模拟键盘输入 + 鼠标移动
 * 使用: CubeMX USB_DEVICE HID配置
 * ============================================ */

#include "main.h"
#include "usb_device.h"
#include "usbd_core.h"
#include "usbd_desc.h"
#include "usbd_hid.h"

extern USBD_HandleTypeDef hUsbDeviceFS;
extern PCD_HandleTypeDef hpcd_USB_FS;

/* HID报表缓冲区 */
uint8_t keyboard_report[8] = {0};
uint8_t mouse_report[4] = {0};

/* 按键状态 */
volatile uint8_t last_key_state = 0xFF;

/* USB HID键盘Key Code定义(HID Usage Table) */
#define KEY_A           0x04
#define KEY_B           0x05
#define KEY_C           0x06
#define KEY_ENTER       0x28
#define KEY_SPACE       0x2C
#define KEY_BACKSPACE   0x2A
#define KEY_TAB         0x2B
#define KEY_LEFT_SHIFT  0x00  /* 修饰键在Modifier字节 */

/* Modifier定义 */
#define MOD_LCTRL       0x01
#define MOD_LSHIFT      0x02
#define MOD_LALT        0x04
#define MOD_LGUI        0x08

/* 函数声明 */
void SystemClock_Config(void);
static void MX_GPIO_Init(void);
static void Check_Buttons(void);
static void Send_Keyboard(uint8_t key, uint8_t modifier);
static void Send_Key_Release(void);
static void Send_Mouse(int8_t x, int8_t y, uint8_t buttons);
static void Send_Mouse_Release(void);

int main(void)
{
    /* HAL初始化 */
    HAL_Init();
    SystemClock_Config();

    /* 初始化GPIO(按键) */
    MX_GPIO_Init();

    /* 初始化USB设备 */
    MX_USB_DEVICE_Init();

    /* 等待USB枚举完成 */
    HAL_Delay(2000);

    printf("USB HID Keyboard/Mouse Ready!\r\n"); /* 通过调试串口 */

    while (1)
    {
        /* 检测按键状态 */
        Check_Buttons();
        HAL_Delay(10); /* 去抖 */
    }
}

/* ============ 按键检测 ============ */
static void Check_Buttons(void)
{
    uint8_t current_state = 0;
    uint8_t changed;

    /* 读取4个按键状态 (0=按下, 1=松开) */
    if (HAL_GPIO_ReadPin(GPIOA, GPIO_PIN_0) == GPIO_PIN_RESET)
        current_state |= (1 << 0);
    if (HAL_GPIO_ReadPin(GPIOA, GPIO_PIN_1) == GPIO_PIN_RESET)
        current_state |= (1 << 1);
    if (HAL_GPIO_ReadPin(GPIOA, GPIO_PIN_2) == GPIO_PIN_RESET)
        current_state |= (1 << 2);
    if (HAL_GPIO_ReadPin(GPIOA, GPIO_PIN_3) == GPIO_PIN_RESET)
        current_state |= (1 << 3);

    changed = current_state ^ last_key_state;

    /* PA0按下 → 输入字母'a' */
    if (changed & (1 << 0))
    {
        if (current_state & (1 << 0))
        {
            Send_Keyboard(KEY_A, 0);
        }
        else
        {
            Send_Key_Release();
        }
    }

    /* PA1按下 → 输入字母'b'(按Shift大写) */
    if (changed & (1 << 1))
    {
        if (current_state & (1 << 1))
        {
            Send_Keyboard(KEY_B, MOD_LSHIFT); /* Shift+B = 'B' */
        }
        else
        {
            Send_Key_Release();
        }
    }

    /* PA2按下 → 鼠标左键点击 */
    if (changed & (1 << 2))
    {
        if (current_state & (1 << 2))
        {
            Send_Mouse(0, 0, 0x01); /* 左键按下 */
            HAL_Delay(50);
            Send_Mouse_Release();  /* 左键释放 */
        }
    }

    /* PA3按下 → 鼠标右键点击 */
    if (changed & (1 << 3))
    {
        if (current_state & (1 << 3))
        {
            Send_Mouse(0, 0, 0x02); /* 右键按下 */
            HAL_Delay(50);
            Send_Mouse_Release();  /* 右键释放 */
        }
    }

    last_key_state = current_state;
}

/* ============ 发送键盘按键 ============ */
static void Send_Keyboard(uint8_t key, uint8_t modifier)
{
    /* 构建键盘报表 */
    memset(keyboard_report, 0, sizeof(keyboard_report));
    keyboard_report[0] = modifier;  /* 修饰键 */
    keyboard_report[1] = 0;           /* 保留 */
    keyboard_report[2] = key;         /* 第一个按键 */
    /* keyboard_report[3~7]可以放其他同时按下的键 */

    /* 通过USB发送 */
    USBD_HID_SendReport(&hUsbDeviceFS, keyboard_report, sizeof(keyboard_report));

    HAL_Delay(10); /* 短暂延时，模拟真实按键节奏 */
}

/* ============ 释放所有按键 ============ */
static void Send_Key_Release(void)
{
    memset(keyboard_report, 0, sizeof(keyboard_report));
    USBD_HID_SendReport(&hUsbDeviceFS, keyboard_report, sizeof(keyboard_report));
}

/* ============ 发送鼠标移动/按键 ============ */
static void Send_Mouse(int8_t x, int8_t y, uint8_t buttons)
{
    mouse_report[0] = buttons;  /* 按钮 */
    mouse_report[1] = (uint8_t)x;  /* X轴移动 */
    mouse_report[2] = (uint8_t)y;  /* Y轴移动 */
    mouse_report[3] = 0;           /* 滚轮 */

    USBD_HID_SendReport(&hUsbDeviceFS, mouse_report, sizeof(mouse_report));
}

/* ============ 释放鼠标按键 ============ */
static void Send_Mouse_Release(void)
{
    memset(mouse_report, 0, sizeof(mouse_report));
    USBD_HID_SendReport(&hUsbDeviceFS, mouse_report, sizeof(mouse_report));
}

/* ============ 自动输入示例(可选) ============ */
void Auto_Type_Demo(void)
{
    /* 自动输入"Hello World!" */
    const char *str = "Hello World!";
    uint8_t i;

    for (i = 0; str[i] != '\0'; i++)
    {
        uint8_t key = 0;
        uint8_t mod = 0;

        /* 简单映射: a-z直接用KEY_A~KEY_Z */
        if (str[i] >= 'a' && str[i] <= 'z')
            key = KEY_A + (str[i] - 'a');
        else if (str[i] >= 'A' && str[i] <= 'Z')
        {
            key = KEY_A + (str[i] - 'A');
            mod = MOD_LSHIFT; /* 大写需要Shift */
        }
        else if (str[i] == ' ')
            key = KEY_SPACE;

        if (key)
        {
            Send_Keyboard(key, mod);
            HAL_Delay(50);   /* 按键按下到释放的间隔 */
            Send_Key_Release();
            HAL_Delay(50);   /* 两次按键之间的间隔 */
        }
    }
}

/* ============ 鼠标自动移动示例 ============ */
void Auto_Mouse_Move_Demo(void)
{
    int8_t i;

    /* 画一个小正方形 */
    for (i = 0; i < 50; i++) Send_Mouse( 5,  0, 0); /* 右移 */
    HAL_Delay(10);
    for (i = 0; i < 50; i++) Send_Mouse( 0,  5, 0); /* 下移 */
    HAL_Delay(10);
    for (i = 0; i < 50; i++) Send_Mouse(-5,  0, 0); /* 左移 */
    HAL_Delay(10);
    for (i = 0; i < 50; i++) Send_Mouse( 0, -5, 0); /* 上移 */
    HAL_Delay(10);
}

/* ============ GPIO初始化 ============ */
static void MX_GPIO_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStruct = {0};

    __HAL_RCC_GPIOA_CLK_ENABLE();

    /* PA0~PA3: 按键输入，内部上拉 */
    GPIO_InitStruct.Pin = GPIO_PIN_0 | GPIO_PIN_1 | GPIO_PIN_2 | GPIO_PIN_3;
    GPIO_InitStruct.Mode = GPIO_MODE_INPUT;
    GPIO_InitStruct.Pull = GPIO_PULLUP;
    HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);
}

/* ============ 系统时钟 ============ */
void SystemClock_Config(void)
{
    RCC_OscInitTypeDef RCC_OscInitStruct = {0};
    RCC_ClkInitTypeDef RCC_ClkInitStruct = {0};

    /* USB需要48MHz时钟(PLL × 1.5) */
    RCC_OscInitStruct.OscillatorType = RCC_OSCILLATORTYPE_HSE;
    RCC_OscInitStruct.HSEState = RCC_HSE_ON;
    RCC_OscInitStruct.PLL.PLLState = RCC_PLL_ON;
    RCC_OscInitStruct.PLL.PLLSource = RCC_PLLSOURCE_HSE;
    RCC_OscInitStruct.PLL.PLLMUL = RCC_PLLMUL_9;    /* 72MHz */
    RCC_OscInitStruct.PLL.PREDIV1 = RCC_PREDIV_DIV1;
    if (HAL_RCC_OscConfig(&RCC_OscInitStruct) != HAL_OK) Error_Handler();

    RCC_ClkInitStruct.ClockType = RCC_CLOCKTYPE_HCLK | RCC_CLOCKTYPE_SYSCLK |
                                  RCC_CLOCKTYPE_PCLK1 | RCC_CLOCKTYPE_PCLK2;
    RCC_ClkInitStruct.SYSCLKSource = RCC_SYSCLKSOURCE_PLLCLK;
    RCC_ClkInitStruct.AHBCLKDivider = RCC_SYSCLK_DIV1;
    RCC_ClkInitStruct.APB1CLKDivider = RCC_HCLK_DIV2;   /* 36MHz */
    RCC_ClkInitStruct.APB2CLKDivider = RCC_HCLK_DIV1;   /* 72MHz */
    if (HAL_RCC_ClockConfig(&RCC_ClkInitStruct, FLASH_LATENCY_2) != HAL_OK)
        Error_Handler();

    /* USB时钟: 必须为48MHz */
    /* F103: USB时钟 = 72MHz / 1.5 = 48MHz (硬件自动) */
}

CubeMX USB HID配置

CubeMX配置步骤:
1. 选择STM32F103C8T6
2. USB_OTG_FS (或USB):
   - Mode: Device (Full Speed)
3. USB_DEVICE:
   - Class for FS IP: Human Interface Device (HID)
   - (如果是复合设备，选Composite Device)
4. 在Middlewares/USB_DEVICE/HID中:
   - IN Endpoint: EP1 IN
   - Packet Size: 8 bytes (HID默认)
5. GPIO:
   - PA0~PA3: GPIO_Input, Pull-Up
   - PA11/PA12: USB DP/DM (自动配置)
6. Clock:
   - 确保48MHz USB时钟
7. NVIC:
   - USB中断: Enabled
8. 生成代码

❓ 常见问题排查

问题1：电脑不识别USB设备

最常见原因：
1. 缺少1.5K上拉电阻(让主机检测到全速设备)
2. USB时钟不是48MHz
3. USB线不支持数据传输(充电线)
4. USB描述符有错误

排查步骤：

  1. 用万用表确认PA12和3.3V之间有1.5K电阻
  2. Windows设备管理器中应该看到"未知USB设备"或"HID设备"

问题2：识别了但按键无反应

原因排查：
1. 报表格式不对 — 键盘报表必须8字节
2. Key Code错误 — 检查是否使用了正确的HID Usage Table键码
3. 按键检测逻辑错误 — 确认GPIO读取正常
4. USB发送间隔太短 — 部分系统需要最小间隔

常用键码速查：

HID Key Codes (部分):
a-z: 0x04-0x1D
1-0: 0x1E-0x27 (与数字键位置对应)
Enter: 0x28
Esc: 0x29
Backspace: 0x2A
Tab: 0x2B
Space: 0x2C
'-': 0x2D, '=': 0x2E, '[': 0x2F, ']': 0x30
';': 0x33, "'": 0x34, '`': 0x35
'\': 0x31, ',': 0x36, '.': 0x37, '/': 0x38
Caps Lock: 0x39
F1-F12: 0x3A-0x45

❓ 常见问题 FAQ

Q: 电脑识别不到USB设备？

A: 1)USB DP需要1.5kΩ上拉到3.3V（全速设备） 2)描述符格式必须严格符合USB规范 3)VBus检测引脚配置 4)时钟精度要求±0.25%

Q: HID报表描述符怎么写？

A: 用USB HID Descriptor Tool生成。初学参考STM32 CubeMX自动生成的模板，不要手写。
