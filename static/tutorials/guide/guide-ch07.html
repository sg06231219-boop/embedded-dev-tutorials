第7章：ESP32开发实战

📖 小白完全指南 · 7/14 | 阅读时间约3分钟

ESP32=单片机界的瑞士军刀——自带WiFi+蓝牙，双核240MHz，520KB RAM，¥10一片。做物联网项目首选。

💡 类比：如果STM32是精工手表（精准、低功耗），ESP32就是智能手机（功能全面、联网、性能强）。做精确控制用STM32，做联网应用用ESP32。

esptool 烧录

# 查看ESP32信息
esptool.py --chip esp32 -p COM3 chip_id

# 烧录固件
esptool.py --chip esp32 -p COM3 -b 460800     write_flash -z 0x1000 bootloader.bin     0x8000 partitions.bin     0x10000 firmware.bin

# 擦除Flash
esptool.py --chip esp32 -p COM3 erase_flash

# 读取Flash
esptool.py --chip esp32 -p COM3 read_flash 0 0x400000 flash_dump.bin

ESP-IDF 基础（官方开发框架）

⚠️ 前提条件：ESP-IDF需要完整安装（约2-3GB），初学者可先跳过，用Arduino框架也能跑ESP32。

Windows安装步骤：
① 下载离线安装包：dl.espressif.com/dl/esp-idf/（推荐，包含所有依赖）
② 双击安装，选择默认路径 C:\Espressif，勾选"添加到PATH"
③ 安装约需10-20分钟，取决于网速
④ 安装完成后，打开"ESP-IDF 5.x CMD"快捷方式（安装器创建的）
⑤ 验证：idf.py --version → 输出版本号即成功

如果离线包下载慢：
• 用手机热点试试（有时比WiFi快）
• 在有VPN的环境下载后拷贝过来
• 用PlatformIO插件（VS Code里装了）创建ESP32项目，无需单独装ESP-IDF
// WiFi扫描示例
#include "esp_wifi.h"
#include "esp_event.h"

void wifi_scan(void) {
    wifi_scan_config_t scan_config = {
        .ssid = NULL,
        .bssid = NULL,
        .channel = 0,
        .show_hidden = true
    };

    esp_wifi_set_mode(WIFI_MODE_STA);
    esp_wifi_start();
    esp_wifi_scan_start(&scan_config, true);  // 阻塞扫描

    uint16_t ap_count = 0;
    esp_wifi_scan_get_ap_num(&ap_count);

    wifi_ap_record_t *ap_list = malloc(sizeof(wifi_ap_record_t) * ap_count);
    esp_wifi_scan_get_ap_records(&ap_count, ap_list);

    for (int i = 0; i < ap_count; i++) {
        printf("%-32s RSSI: %d Ch: %d
",
            ap_list[i].ssid, ap_list[i].rssi, ap_list[i].primary);
    }
    free(ap_list);
}

Arduino vs ESP-IDF

特性ArduinoESP-IDF
上手难度极低中等
灵活性受限完全控制
性能优化一般极致
FreeRTOS隐藏使用显式使用
适合原型验证/初学者产品开发/进阶
💡 建议：Arduino快速验证想法，ESP-IDF做正式产品。PlatformIO支持两种框架切换。

⚡ ESP32 5分钟快速上手（Arduino框架）

⚠️ 最快的方式：用PlatformIO创建ESP32 Arduino项目，无需单独装ESP-IDF！
步骤1：VS Code → PlatformIO → New Project → Board: ESP32 Dev Module → Framework: Arduino
步骤2：把下面的代码粘贴到 src/main.cpp
// ESP32 LED闪烁 + 串口输出（Arduino框架，5分钟跑通）
// 大多数ESP32开发板的LED在GPIO2

void setup() {
    Serial.begin(115200);  // 初始化串口
    pinMode(2, OUTPUT);     // GPIO2设为输出
    Serial.println("ESP32 Started!");
}

void loop() {
    digitalWrite(2, HIGH);   // LED亮
    Serial.println("LED ON");
    delay(500);
    digitalWrite(2, LOW);    // LED灭
    Serial.println("LED OFF");
    delay(500);
}
步骤3：连接ESP32 → 点击PlatformIO的→箭头按钮（Upload） → 自动编译+烧录
步骤4：打开串口监视器（PlatformIO插头图标），看到"LED ON/OFF"交替输出就成功了！
🔍 常见问题：esptool连接失败
1. 按住BOOT键再点RST键，进入下载模式
2. 检查COM口号是否正确
3. 某些ESP32板需要手动按BOOT进入下载模式
4. USB驱动：CP2102/CH340确认已安装
